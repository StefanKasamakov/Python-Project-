Проект: Складова наличност

Описание:
Този проект е направен на Python и служи за управление на складова наличност.

Функции:
- Добавяне на продукт
- Показване на продукти
- Проверка за наличност

Файлове:
- models.py -> съдържа класа Product
- main.py -> главна програма
- requirements.txt -> използвани библиотеки

Стартиране:
python main.py
